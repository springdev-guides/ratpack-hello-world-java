= Ratpack: 编写一个 Hello World 应用（Java）
rain
2019-01-17

本文介绍如何使用 Ratpack 创建一个 Web 应用，使用 Java 语言编写一个简单的 “Hello World”，使用 Gradle 构建并运行起来。

Ratpack 是一组 Java 库，可以促进快速、高效、可演化和经过良好测试的 HTTP 应用程序。
它建立在高性能和高效的 Netty 事件驱动的网络引擎上。
